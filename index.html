Pengertian HTML

HTML atau singkatan dari Hypertext Markup Language adalah bahasa markup yang digunakan untuk membuat dan mendesain halaman web. HTML memungkinkan pengguna untuk menambahkan konten seperti teks, gambar, video, audio, dan elemen-elemen lain ke halaman web dan juga membuat struktur dan tata letak dari halaman tersebut. HTML adalah salah satu teknologi dasar yang digunakan dalam pengembangan web. Halaman web yang dibuat dengan HTML dapat ditampilkan pada berbagai perangkat seperti komputer, tablet, dan smartphone. HTML juga mendukung berbagai fitur seperti hyperlink, form input, table, dan multimedia.

Text Formating

  1. Bold text adalah teks tebal.
  2. Italicized text adalah teks miring.
  3. Underlined text adalah teks bergaris bawah.
  4. Strikethrough text adalah teks yang dicoret.
  5. Superscript text adalah teks superskrip.
  6. Subscript text adalah teks subscript.
